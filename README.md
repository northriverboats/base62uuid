Base62Uuid
==========
Will generate 12 digit long random UUIDs

# Installing
```
git clone git@github.com:northriverboats/base62uuid.git
cd base62uuid
python3 -m venv venv
source venv/Scripts/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

# running
```
cd base62uuid
./base62uuid.py
```
